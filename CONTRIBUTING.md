# Contributing to OS-Lab

Welcome! We appreciate your interest in contributing to the **Operating Systems Lab Manual**!

## How to Contribute
1. **Fork** the repository.
2. **Clone** your fork.
3. **Create a Branch** for your feature or fix.
4. **Make Changes** and follow the style.
5. **Commit** with meaningful messages.
6. **Open a Pull Request**.

## Project Guidelines
- Code should be clean and well-documented.
- Maintain consistency with the current directory structure.
- Add output files for any new experiment.

---
Thank you for making OS-Lab better! 🐧
